# 📖 About

This project is a **Python-based Feedback Sentiment Analysis System** that analyzes customer feedback using keyword matching.

The program classifies feedback into:

- ✅ Positive
- ❌ Negative
- ⚪ Neutral

It also displays sentiment statistics and provides a keyword search feature for finding specific feedback.

---

# 🚀 Features

✅ Accepts multiple user feedbacks

✅ Cleans text automatically

✅ Detects Positive feedback

✅ Detects Negative feedback

✅ Detects Neutral feedback

✅ Displays sentiment counts

✅ Shows complete feedback lists

✅ Keyword Search functionality
